## 自定义View简述
* 自定义View属性
* 在构造方法中获得自定义的属性
* 重新ononDraw：用来绘制该控件的内容
* 重新onMeasure：用来确定该控件或子控件的大小
* 重新onLayout：用来确定该控件及子控件在父窗体中的位置
